Description

Template Blocks is a visual template engine, created with the logic of the web designer in mind and is aiming to help you create web templates easier and faster.

If you're using Template Blocks for the first time, be sure that your server meets the minimum requirements (PHP 5, MySQL) and that you have already created the database you are going to use for this engine.

Being this an early version it may be rough around the edges and a set of features and improvements are left to be included in following releases.

Having said that, it is a functional template engine which we hope you enjoy using and embrace it as a useful tool for your everyday web designing.

 

Instructions

  • Installing: It should be as simple as uploading the files on your server and running the setup process.
  • Updating: Copy all new files in your template folder EXCEPT the "assets" folder & the "config.php" file. Also be careful when copying files in the "classes" folder if you have created custom classes yourself.

In any case you will need to initiate the setup process. To do that just open your web browser and load the domain of your website.

After the setup is complete you can log in to the administration interface at:
http://www.yourdomain.com/template/admin.php

 

Support

There are a number of things you can do beyond the basic setup.

- Download a variety of blocks, created and uploaded by other users to use on your site.

- Read the FAQs for a series of common questions answered.

- Want a feature added to the template engine? Submit a comment to the changelog.

If you need further assistance please send your notes to info@templateblocks.com
